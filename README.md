# bun-svelte-kit

## Description

Template project for **Bun** + **Svelte Kit**

## Usage

### Setup

```bash
bun install
```

### Dev

```bash
bun run dev
bun run format
bun run lint
```

### Prod

```bash
bun run build
bun run start
```

## Dependencies

### Svelte

- svelte
- @sveltejs/kit
- svelte-adapter-bun
- vite

### Styling

- tailwindcss
- @tailwindcss/vite

### Linting

- eslint
- @eslint/js
- typescript-eslint
- eslint-plugin-svelte
- eslint-config-prettier

### Formatting

- prettier
- prettier-plugin-svelte
- prettier-plugin-tailwindcss

### Utils

- @types/bun
- typescript
