# bun-svelte-kit

## Description

Template project for **Bun** + **Svelte Kit**

## Usage

```bash
bun install
bun --bun run svelte-kit sync
bun run dev
bun run build && bun run start
```

## Dependencies

### Svelte

- svelte
- @sveltejs/kit
- svelte-adapter-bun
- vite

### Styling

- tailwindcss
- @tailwindcss/vite

### Linting

- eslint
- @eslint/js
- typescript-eslint
- eslint-plugin-svelte

### Formatting

- prettier
- prettier-plugin-svelte
- prettier-plugin-tailwindcss

### Utils

- @types/bun
- typescript
